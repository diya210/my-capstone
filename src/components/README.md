# My Capstone Project - Solar System Explorer

## Project Summary
A React web application that displays information about planets 
in our solar system using dynamic data, built as a capstone project.

## Live Demo
https://clinquant-genie-21b7f6.netlify.app

## Group Members
- Kay - [GitHub](https://github.com/diya210)

## Components
- Header - Hero section with navigation
- DataSection - Video and data explanation
- PlanetGrid - Fetches and displays planet data
- TableSection - Planetary facts table
- ContactForm - Contact form with validation
- Footer - Links and team info

## How to Run Locally
1. Clone the repo: `git clone https://github.com/diya210/my-capstone.git`
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`
